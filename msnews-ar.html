MSNewsAR - Augmenting Web Browsing Experience

Microsoft Azure Prize Champion at nwHacks 2019. A mobile app for classifying images using AI, discovering related online videos and web pages, and rendering them in augmented reality.

Video Demo

Overview of the App

MSNewsAR, an Augmented Reality (AR)-based news browser, has been honored with the Best Project Award from Microsoft Azure at the nwHacks hackathon.

The robust backend was developed using Python and Flask. Its main functionalities include scraping core details from web pages and using the Bing search engine for deep analysis of static images on these webpages to fetch related information and videos. Furthermore, Flask was integrated with Docker and deployed on the Microsoft Azure Container Instances (ACI). Simultaneously, using iOS ARKit and Swift, the backend cached web page information is extracted and then real-time localized rendering of image links and video content is done in an AR environment.
